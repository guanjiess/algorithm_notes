1、统计连续子序列的长度

```C++
string s;
int cnt = 0;
unordered_mp<char, int> mp;
for(int i = 0; i < s.size(); i++){
    char c = s[i];
    ++cnt;
    if(i ==  s.size()-1 || c != s[i+1]){
        mp[c] = cnt;
        cnt = 0;
    }
}
```
